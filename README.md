# Smart Lens

## A easy way to find files using images and content instead of remembering names and file locations.

First clone the repository:

```
git clone https://github.com/yourusername/smart-lens.git
```

Then navigate to the project directory:

```
cd smart-lens

```

Install all the dependencies:

```
npm install
```

Now install the cli:

```
npm install -g .
```
